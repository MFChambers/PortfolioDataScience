Maia Chambers – Data Science Portfolio

This portfolio highlights a compilation of applied data science projects—cleaning, analysis, visualization, and modeling—using real-world datasets to uncover insights with Python, SQL, and data visualization.

Student project - Using Palmer Penguins data

Introduced foundational data science workflows using Quarto and Python. I explored the Palmer Penguins dataset by creating tables and visualizations from the Python for Data Science (PY4DS) textbook.

Client Project 1 — Historical and Cultural Patterns in Baby Names

Analyzed U.S. baby name trends using Python (pandas, seaborn) and Quarto to explore historical, cultural, and pop-culture influences on naming patterns.

Client Project 2 – Finding Relationships in Baseball

Explored baseball performance using SQL + Quarto; surfaced player/game insights.

Client Project 3 – Flight Delay Analysis

Cleaned and analyzed U.S. flight delay data (JSON) with pandas and Lets-Plot, identifying the worst airports, best travel months, and weather’s role in disruptions. pandas-based analysis of delay drivers; trends and contributing factors.

Client Project 4 – Predicting Pre-1980 Homes

Used ML models (Decision Tree, Random Forest, Logistic Regression) to predict whether homes were built before 1980.

Client Project 5 – Income Prediction & Data Validation

Cleaned and validated the FiveThirtyEight Star Wars survey; engineered features and trained a Random Forest to predict >$50k income (~75.7% accuracy) using pandas, scikit-learn, and Matplotlib. Full Quarto report with statistical analysis and visualization.

Coding Challenge – Timed

Timed Python challenge: loaded public datasets, built quick visualizations, engineered features, and trained a Random Forest baseline with feature importances.
