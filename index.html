Student project - Using Palmer Penguins data

Introduced foundational data science workflows using Quarto and Python. I explored the Palmer Penguins dataset by creating tables and visualizations from the Python for Data Science (PY4DS) textbook. The project demonstrates how to build reproducible reports, cleanly display tabular data, and use visual encodings (color, shape, smoothing lines) to highlight relationships—in this case, showing that penguins with longer flippers generally weigh more.
